# Northwind Goods — Checkout

A checkout for a small homewares shop, put together quickly with AI help. We use it as the starting point for a short, live design conversation.

## 1. Get it running

Fork this repo, clone your fork, then:

```bash
bun install      # or: npm install
bun dev          # or: npm run dev
```

Open the local URL at `http://localhost:5173`

## 2. Look it over

Go through the page the way you would if a teammate asked you to review their work before it shipped. We mostly care about your **eye for detail and your reasoning**, not how much you can fix in the time we have.

- What would you change, and why? Visual, UX, accessibility, component choices, copy, anything that stands out.
- What would you **leave alone**? Calling out what's already fine is just as useful.
- What's the priority order? You won't get to everything.

## 3. Propose fixes

Make a branch and capture the changes you'd make. **Plain-English descriptions are completely fine** . E.G. "the order total should stand out more; make it larger and a darker shade." Actual code changes are welcome too if time allows, but they're not the point.

You can use Claude (or any AI) to draft the fixes from your descriptions. No AI plan of your own? No problem. Tell me what you'd prompt and we'll run it on a shared screen.

We'll do this together live and talk through your thinking, so there's nothing to prepare. Ask questions any time.
